# Tennis Match Manager App

##  概要
大学テニスサークルでの練習運営を効率化、かつ、テニスの新しい体験価値を付与するために開発した  ダブルス対戦表生成＆スコア管理アプリ。

対戦作成の手間削減と、個人成績の可視化を目的としている。

---

##  開発背景

実際の練習現場で以下の課題があった：

- 毎回ホワイトボードで対戦表を作るのに時間がかかる
- スコアが記録されず、成績が蓄積されない
- 練習会ごとにデータを分離できない

これらを解決するため、アプリとして設計・実装。

---

##  設計の特徴

### ① セッション単位の状態管理

- `Session`クラスを導入
- 参加者・試合・スコアをセッションに集約
- 画面(State)とデータを分離

 UIは「表示装置」、データは「Sessionが保持」という設計

---

### ② オブジェクト指向設計

- Matchクラス
- Sessionクラス
- 状態とロジックのカプセル化
- getterによる安全な値参照

---

### ③ 将来拡張を見据えた構造

- SQLite保存を想定したデータ設計
- セッション履歴の永続化を想定
- Firebase連携を視野に入れた構成
