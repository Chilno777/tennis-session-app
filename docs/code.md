〇getterとは
・getにすることで、関数なのに変数みたいに扱うことができる。
・計算結果をプロパティのように扱うことができる
・メリット：UIをきれいに出すことができる
メソッド（Method）はオブジェクトの「動作・処理（何をするか）」、プロパティ（Property）はオブジェクトの「状態・情報（属性・どんなものか）」を指す、オブジェクト指向プログラミングの基本概念です。車で例えると、プロパティは「色・速度」、メソッドは「走る・止まる」にあたります

〇session（練習会）について
セッションに関係あるデータと処理を、すべてsessionクラスに持たせる

「練習会（Session）」って何を持つべき？

参加者、試合一覧、試合順、スコア、成績集計、現在のカーソル

この情報を画面側が持っていると、

画面を離れたら消える、別の練習会を作れない、テストしにくい、ロジックがUIに混ざる

要するに、セッションという箱の中に、責任をひとまとめにした状態

メリット
画面が変わってもデータが残る、セッションを複数持てる
